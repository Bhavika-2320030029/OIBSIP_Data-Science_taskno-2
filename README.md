TASK 2: UNEMPLOYMENT RATE ANALYSIS

Repository Name: Unemployment-Rate-Analysis


 Project Title:

Unemployment Rate Analysis During COVID-19


 Problem Statement:

Unemployment rate is defined as the percentage of unemployed individuals in the total labor force.
During the COVID-19 pandemic, there was a sharp increase in unemployment rates.

This project analyzes unemployment trends and visualizes the impact of COVID-19.

 Objectives:

To analyze unemployment rate data

To visualize trends before and during COVID-19

To extract meaningful insights from the dataset


Dataset Description:

The dataset includes

Region / State

Date

Unemployment Rate

Employment Rate

Labour Participation Rate


 Methodology:

Load and clean the dataset

Perform exploratory data analysis

Visualize unemployment trends

Compare pre-COVID and COVID periods

Interpret insights


 Visualizations Used:

Line charts

Bar plots

Correlation heatmaps


 Key Insights:

Unemployment rose significantly during COVID-19

Gradual recovery observed post-pandemic

Regional variations in unemployment rates


 Tools & Technologies:

Python

Google Colab

Pandas, NumPy

Matplotlib, Seaborn


 Conclusion:

The analysis highlights the economic impact of COVID-19 and demonstrates how data analysis helps in understanding real-world economic challenges.
